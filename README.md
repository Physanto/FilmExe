# 🎬 Gestión de Cines
- Descripción

Este proyecto es una aplicación sencilla para la gestión básica de un cine. Permite manejar información como películas, salas y funciones, utilizando una base de datos para almacenar los datos.

Está desarrollado siguiendo el patrón de arquitectura MVC (Modelo - Vista - Controlador), separando la lógica del programa, la interfaz y el manejo de datos.

Su objetivo principal es servir como práctica para entender cómo estructurar un proyecto y trabajar con bases de datos de forma organizada.
